# 🚀 AI 성취도 예측 및 학습 최적화 시스템 구축 플랜

이 문서는 '스윗미' 프로젝트에 AI 기반의 성취도 예측, 맞춤형 피드백, 그리고 지능형 학습 계획 추천 기능을 추가하기 위한 상세 실행 계획을 담고 있습니다.

---

## 📋 개요 (Overview)
단순한 데이터 기록을 넘어, 축적된 학습 데이터를 분석하여 유저에게 **미래의 성적을 예측**해주고, **최적화된 공부 방향을 제시**하는 AI 어시스턴트 기능을 구현합니다.

---

## 🛠 주요 기능 및 구현 단계 (Main Features & Phases)

### Phase 1: AI 성취도 예측 엔진 (Score Prediction)
유저의 최근 학습 일관성과 성과 지표를 바탕으로 예상 성적(Index)을 산출합니다.

*   **입력 데이터**: 최근 14일간의 `reflection` 데이터 (달성률, 시간 관리, 오답 정리, 복습 여부).
*   **예측 로직**:
    *   `Base Score`: 현재까지의 평균 성취도 점수.
    *   `Consistency Multiplier`: 공부 시간의 표준 편차가 낮을수록(일관될수록) 가점.
    *   `Review Weight`: 복습 횟수가 망각 곡선 주기에 가까울수록 가점.
*   **출력**: "현재 습관 유지 시 다음 시험 예상 점수: **85점**"

### Phase 2: 지능형 리마인드 및 피드백 (Adaptive Feedback)
망각 곡선과 학습 누락 데이터를 감지하여 유저에게 잔소리가 아닌 '데이터 기반 조언'을 제공합니다.

*   **피드백 로직**:
    *   **망각 감지**: 특정 단어/개념 복습 후 7일 이상 경과 시 알림.
    *   **취약점 경고**: 특정 과목의 오답 정리(wrong) 점수가 지속적으로 낮을 때 "이 과목은 지금 보충이 필요해요!" 팝업.
*   **UI/UX**: 분석 탭 상단에 'AI의 오늘의 메시지' 카드 배치.

### Phase 3: AI 기반 데일리 공부 플랜 추천 (AI Planner)
유저의 과거 패턴을 분석하여 내일의 스케줄을 제안합니다.

*   **추천 알고리즘**:
    *   **과목 밸런싱**: 최근 3일간 공부 시간이 가장 적었던 과목 우선 배치.
    *   **피크 타임 활용**: 유저가 가장 집중력이 높았던 시간대(타이머 로그)에 핵심 과목 배치.
    *   **Buffer Time**: 미달성 확률을 고려하여 계획 사이에 15%의 여유 시간 강제 삽입.

### Phase 4: UI/UX 고도화 (Advanced Visualization)
데이터를 시각적으로 이해하기 쉽게 표현합니다.

*   **성취도 예측 그래프**: 현재 추세선과 목표 점수 사이의 간극 시각화.
*   **학습 밀도 히트맵**: 일별 몰입도(순공 시간 + 성취도)를 색상 농도로 표현.

## 🎯 기대 효과 (Expected Outcome)
1.  **학습 동기 부여**: 예상 점수가 가시화됨으로써 유저가 공부량을 능동적으로 조절하게 됨.
2.  **메타인지 강화**: 자신의 망각 주기와 취약점을 객관적인 수치로 확인 가능.
3.  **계획 수립 피로도 감소**: AI가 제안하는 기본 틀을 바탕으로 효율적인 계획 수립.
