# Icon Reference

> 프로젝트에서 사용하는 모든 아이콘의 명세.  
> 기능 추가/수정 시 이 문서를 참조하여 일관된 아이콘을 사용할 것.

---

## 아이콘 시스템 개요

- **소스**: Lucide v1.8.0 기반 인라인 SVG (`js/icons.js`)
- **CDN 의존 없음**: 오프라인(PWA) 완벽 지원
- **사용법**: JS 동적 콘텐츠에서는 `icon(name, size?, extraStyle?)` 함수 사용
- **정적 HTML**: `<i data-lucide="icon-name"></i>` 작성 후 `initStaticIcons()`가 초기화 시 1회 변환

```js
// 기본 사용
icon('trash-2')             // 16px 기본 크기
icon('sparkles', 14)        // 14px 크기
icon('zap', 14, 'fill: currentColor;')  // 커스텀 스타일
```

## 새 아이콘 추가 규칙

1. **`js/icons.js`의 `SVG_PATHS` 객체에 추가** — Lucide 아이콘 기준 `<svg>` 태그 내부의 path/rect/circle 등만 기입
2. **이 문서(`icon-reference.md`)에 등록** — 키, 모양 묘사, 용도, 사용 위치 명시
3. **기존 아이콘과 용도가 겹치지 않는지 확인** — 동일 용도에는 동일 아이콘 사용
4. SVG 데이터는 [Lucide Icons](https://lucide.dev/icons/) 에서 확인 가능

---

## 주의사항

- 동적 HTML(template literal)에서는 반드시 `icon()` 함수를 사용할 것 (`<i data-lucide>` 사용 금지)
- 정적 HTML에서는 `<i data-lucide="..."></i>` 사용 가능 (`initStaticIcons()`가 초기화 시 변환)
- `refreshIcons()`는 제거됨 — 동적 콘텐츠에 `icon()` 사용으로 불필요
