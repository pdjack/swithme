## 0. 현재 상태 요약

| 항목 | 상태 |
|---|---|
| 전체 진행 | ☐ 착수 전 (0/4 단계) |
| **스택 확정** | Firebase(Auth·Firestore·FCM) + Google Play IAP — PRD §9-5 |
| 목적 유의 | **학습**. 결제는 샌드박스, 실수익 없음 (PRD §10-2) |
| 순서 제약 | ①계정이 뿌리 → ②동기화 → ③푸시 / ④결제(②·계정 후) |
| 현재 병목 | **사전 준비 P** — 유저 구글 계정으로 Firebase 프로젝트 생성(유저 몫) |

### 이미 확보된 것

- PWA 매니페스트·Service Worker·아이콘 (TWA 재료와 공유)
- localStorage `state` 중앙 관리 구조 (동기화 대상 원본)

## P. 사전 준비 (유저 몫 — 이거 없으면 착수 불가)

- [ ] ☐ **P-1** Firebase 콘솔에서 프로젝트 생성 (유저 구글 계정)
- [ ] ☐ **P-2** 웹 앱 등록 → 설정 키 확보
- [ ] ☐ **P-3** Firebase Auth에서 구글 로그인·이메일 로그인 사용 설정
- [ ] ☐ **P-4** 무료 티어(Spark) 한도 확인 — 학습 트래픽은 초과 안 함

> 설정 키는 시크릿이므로 `.env`로 관리한다(코드 하드코딩 금지).
