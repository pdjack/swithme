# 스윗미 스마트 가이드 (Smart Guidance)

이 가이드는 스윗미의 데이터 분석 결과에 따라 사용자가 즉각적으로 실행할 수 있는 생산성 전략을 제공합니다.

---

## 1. 패턴 데이터 기반 맞춤형 전략

### 몰입 밀도 저하 시: 집중 세션 전환
- **분석 결과**: 활동 시간 대비 실질적인 성취감이 낮거나 멍하니 있는 시간이 많을 때.
- **실행**: 현재 카테고리의 활동 방식을 점검하고, 짧고 집중적인 세션(25~30분)으로 전환하세요.

### 정체기 극복: 카테고리 교차 (Interleaving)
- **분석 결과**: 한 카테고리만 오래 붙잡고 있어 집중 효율이 급격히 떨어지는 시점.
- **실행**: 같은 카테고리만 반복하기보다 성격이 다른 2~3개 카테고리를 섞어서 진행하세요. 뇌의 유연성을 높여 효율을 극대화합니다.

---

## 2. 대시보드 데이터 활용 루틴

### 목표 달성률 최적화
- **Tips**: 수립한 계획의 실천율이 70% 미만이라면, 마감일로부터 역순으로 분량을 재조정하세요. 무리한 계획보다 완수 가능한 '작은 성공'의 반복이 중요합니다.

### 시간 관리의 함정
- **Tips**: 타이머 시간만 채우는 비효율적 활동을 주의하세요. 회고에서 집중도와 몰입도를 함께 기록하여, 가장 효율적인 시간대를 찾아내야 합니다.

---

## 3. 생산성 최적화 상식
- **메타인지 강화**: 내가 잘하는 것과 부족한 것을 명확히 구분하는 것이 개선의 시작입니다. 회고 점수가 지속적으로 낮은 카테고리를 우선 점검하세요.
- **환경 변수 통제**: 데이터 분석 상 특정 환경이 집중도를 낮추는 패턴이 확인된다면 환경을 조정한 후 집중 모드를 활용하세요.
