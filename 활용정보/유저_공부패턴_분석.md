# 🧠 스윗미 데이터 분석 철학 (Analysis Philosophy)

스윗미의 '분석' 기능은 단순한 통계 제공을 넘어, 유저의 고유한 학습 패턴을 진단하고 성공을 위한 명확한 처방전을 제시하는 데 목표가 있습니다.

---

## 1. 핵심 분석 로직

### 진단 레벨: 메타인지 패턴 (Recognition Pattern)
- **지표**: 과목별 목표 달성률 + 오답 정리 점수 + 회고 점수.
- **처방**: 단순히 시간을 많이 보낸 과목보다 '지식의 구멍(오답)'을 메우는 활동에 집중하고 있는지 분석하여 "약점 보완 필요" 경고 혹은 "안정 궤도" 진단을 내립니다.

### 진단 레벨: 몰입 에너지 (Focus Density)
- **지표**: 타이머 작동 시간 vs 휴식 간격 vs 뇌 피로도.
- **처방**: '순수 공부 시간'을 계산하여 현재 유저가 몰입 중인지, 혹은 기력이 소진되어 효율이 낮은 상태인지를 시각화합니다. (에너지 고갈 시 '휴식 권장' 제언)

### 진단 레벨: 루틴 일관성 (Routine Consistency)
- **지표**: 공부 시작 시각의 표준 편차 + 주간 실천율 변동성.
- **처방**: 불규칙한 생활 패턴이 학습 효율 저하의 원인인지를 진단합니다.

## 2. 유형별 AI 자동 리포트 시나리오

| 발견된 패턴 | 분석 진단 | 스윗미의 추천 (Solution) |
| :--- | :--- | :--- |
| **시간은 많으나 성과 저조** | "가짜 공부" 확률 높음 | 타이머 중단 후 30분간 '인출 연습' 세션 유도 |
| **특정 요일/시간대 하락** | 환경적 집중 저해요인 발생 | 해당 시간대에 '교차 학습'으로 주의 환기 제안 |
| **계획 미달성 지속** | 무리한 역치 설정 | 주간 계획 15% 감축 및 Buffer Day 확보 권장 |
| **복습 데이터 부재** | 망각 가속화 상태 | 최근 학습 과목에 대한 '복습 타이머' 즉시 가동 |

## 3. 차세대 지능형 기능 가이드
- **성적 예측 모델**: 현재의 실천율과 누적된 오답 데이터를 기반으로 가상의 시험 성취도를 추론하여 동기 부여를 강화합니다.
- **맞춤형 모드 추천**: 유저의 집중 패턴에 따라 '몰입 모드(Zen)'와 '회고 모드'를 지능적으로 실시간 추천합니다.
- **집중 환경 상관분석**: 외부 데이터(음악, 식단 등)와 학습 성과의 상관관계를 분석하여 유저만의 '최적 학습 환경' 리포트를 발행합니다.
