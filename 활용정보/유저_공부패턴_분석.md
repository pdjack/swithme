# 스윗미 데이터 분석 철학 (Analysis Philosophy)

스윗미의 '분석' 기능은 단순한 통계 제공을 넘어, 유저의 고유한 활동 패턴을 진단하고 개선을 위한 명확한 처방전을 제시하는 데 목표가 있습니다.

---

## 1. 핵심 분석 루프

```
계획 수립 → 실행(타이머) → 회고 기록 → 패턴 분석 → 맞춤 조언
     ↑                                              │
     └──────────────────────────────────────────────┘
```

유저가 직접 축적한 회고 데이터를 기반으로 분석합니다. 별도의 성향 테스트 없이, 실제 행동 데이터에서 패턴을 도출합니다.

---

## 2. 분석 결과 구조

> 각 항목 옆 `✅`는 구현 완료, `☐`는 미구현/예정. (CLAUDE.md "기획·철학 문서의 각 항목에 구현 상태 마커를 표기할 것" 규칙 적용)

### 2-1. 종합 점수 ✅

- 선택한 기간(7일/30일)의 회고 점수 평균을 산출합니다.
- ✅ 직전 동일 기간 대비 증감(▲/▼/–)도 함께 표시합니다.

### 2-2. 카테고리별 트렌드 그래프 ✅

- 각 카테고리의 활동 시간이 시간 흐름에 따라 어떻게 변화하는지 시각화합니다. (현재 구현은 회고 점수가 아닌 **카테고리별 활동 시간(분)** 기준)
- 상승/하락/정체 추세를 한눈에 파악할 수 있습니다.

### 2-3. 인사이트

트렌드 데이터에서 도출한 패턴을 자연어로 제시합니다.

- ✅ 요일별 변동 패턴 (예: "화~목 집중도가 높고 금요일에 급락")
- ☐ 카테고리 간 상관관계 (예: "운동 점수가 높은 날 업무 집중도도 상승")
- ✅ 상승/하락 추세 감지 (선형 회귀 기울기 ±0.5 임계)
- ✅ 미활동 카테고리 경고
- ✅ 회고 기록률 저하 경고

### 2-4. 처방전 ✅

종합 점수·트렌드·인사이트를 종합하여 구체적인 행동 방안을 제시합니다.

---

## 3. 진단 레벨

### 몰입 에너지 (Focus Density) ☐
- **지표**: 타이머 작동 시간 vs 휴식 간격.
- **처방**: 순수 활동 시간을 계산하여 현재 유저가 몰입 중인지, 기력이 소진되어 효율이 낮은 상태인지를 시각화합니다.

### 루틴 일관성 (Routine Consistency) ☐
- **지표**: 활동 시작 시각의 표준 편차 + 주간 실천율 변동성.
- **처방**: 불규칙한 생활 패턴이 효율 저하의 원인인지를 진단합니다.

---

## 4. 패턴별 자동 리포트 시나리오

| 상태 | 발견된 패턴 | 분석 진단 | 스윗미의 추천 (Solution) |
| :--- | :--- | :--- | :--- |
| ☐ | **시간은 많으나 성과 저조** | 비효율적 활동 확률 높음 | 활동 방식 점검 및 집중 세션 유도 |
| ☐ | **특정 요일/시간대 하락** | 환경적 집중 저해요인 발생 | 해당 시간대에 카테고리 전환으로 주의 환기 제안 |
| ☐ | **계획 미달성 지속** | 무리한 목표 설정 | 주간 계획 15% 감축 및 Buffer Day 확보 권장 |
| ☐ | **특정 카테고리만 지속 하락** | 해당 영역 동기 저하 | 목표 재설정 또는 접근 방식 변경 제안 |

---

## 5. 차세대 기능 가이드
- ☐ **맞춤형 모드 추천**: 유저의 집중 패턴에 따라 '몰입 모드(Zen)'와 '회고 모드'를 지능적으로 실시간 추천합니다.
- ☐ **환경 상관분석**: 외부 데이터(음악, 식단 등)와 활동 성과의 상관관계를 분석하여 유저만의 '최적 환경' 리포트를 발행합니다.
